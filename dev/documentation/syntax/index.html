Gear Syntax Reference

Comments

Single line comment

Single line comments begin with a # and end with a new line.

# This is a single line comment.
Multi-line comment

Multi-line comments can nest.

#{ This is a
   multi-line comment #}

Numbers

Integers

Can be written in base 2, base 8, base 10, and base 16.

0b1010
0c217
12345
0x91FF
Fractionals

Can be written with a decimal or in scientific notation.

3.14159
1.54334E-34
Arithmetic

The basic arithmetic operations are available.

59 + 183 - 247
360 * 3.14159 / 180.0

Booleans

Booleans

All values in Gear are truthy except for false and nil.

true
false

Strings

Single Line

Strings are immutable. There is no difference between single and double quoted strings other than personal preference.

'This is a single quoted string'
"This is a double quoted string"
Escaped

Many of the same escape sequences in C are present in Gear.

"This is \n an \"escaped\" \t string"
Interpolation

Strings can evaluate expressions between the ${ and }.

"The sum of 2 + 2 is ${2 + 2}"
Literal Strings

Strings prefixed with an @ sign are treated "as-is" and are unescaped and do not perform string interpolation.

@"Contains \n a literal backslash n"

List

Creation

Lists are ordered sequences of valuess. They can be written with bracket literals and can contain any mix types.

[3.14159, true, 'Hello World']
Insertion

Append a value with "push", prepend with "shift", and insert at index N with "insert".

let fruit = List new()
fruit push('apple')
fruit shift('grape')
fruit insert(2, 'watermelon')
Lookup

Lists index from one where index 1 is the first value and index -1 is the last element.

let fruit = ['apple', 'grape', 'watermelon']
let apple = fruit[1]
let watermelon = fruit[-1]
