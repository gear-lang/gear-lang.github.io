void gear_assign_new_list_with_length(gear_state* state, gear_variable variable, int length)

Creates a nil-populated list of size N and assigns it to a variable.

Parameters
state in The runtime context.
variable out The variable to assign the list to.
length in The initial number of nil elements to create the list with.

This function is recommended when you know the length you need for the list. The reason is because creating a list with gear_assign_new_list and appending onto it may trigger multiple resize allocations which is less optimal.

Since

0.1.0
