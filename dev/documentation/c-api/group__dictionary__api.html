Function documentation

void gear_dict_set_value(gear_state* state, gear_variable dict, gear_variable key, gear_variable value)

Retrieves the value of a dictionary entry.

Parameters
state in The runtime context.
dict in A variable whose value is a dictionary.

If this is not a dictionary, then the last error is set to GEAR_INVALID_VALUE.
key in A variable whose value is a key to associate with the given value.
value in A variable whose value will be associated with the given key.

If an out-of-bounds index is given, then the last error is set to GEAR_INVALID_OPERATION.

Since

0.1.0

void gear_dict_get_value(gear_state* state, gear_variable dict, gear_variable key, gear_variable output)

Retrieves the value of a dictionary entry.

Parameters
state in The runtime context.
dict in A variable whose value is a dictionary.

If this is not a dictionary, then the last error is set to GEAR_INVALID_VALUE.
key in A variable whose value is a key to lookup.
output out A variable to assign the value of the dictionary element at the given index.

If an out-of-bounds index is given, then the last error is set to GEAR_INVALID_OPERATION.

Since

0.1.0

gear_bool gear_dict_has_key(gear_state* state, gear_variable dict, gear_variable key)

Determines if a key is used by a dictionary.

Parameters
state in The runtime context.
dict in A variable whose value is a dictionary.

If this is not a dictionary, then the last error is set to GEAR_INVALID_VALUE.
key in A variable whose value is the key to check for in the dictionary.
Returns #gear_true if the key is present in the dictionary otherwise #gear_false.

Since

0.1.0

int gear_dict_get_count(gear_state* state, gear_variable dict)

Returns the number of elements in a dictionary.

Parameters
state in The runtime context.
dict in A variable whose value is a dictionary.

If this is not a dictionary, then the last error is set to GEAR_INVALID_VALUE.
Returns The number of elements in the dictionary. The value will never be negative.

Since

0.1.0

void gear_dict_for_each(gear_state* state, gear_variable dict, gear_for_each_pair callback, void* user_data)

Iterates over each key/value pair of a dictionary.

Parameters
state in The runtime context.
dict in A variable whose value is a dictionary.

If this is not a dictionary, then the last error is set to GEAR_INVALID_VALUE.
callback in A function to invoke for each key/value pair of the dictionary.
user_data in A user pointer to give to the callback function; this can be NULL.
