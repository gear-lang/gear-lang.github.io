void gear_list_append(gear_state* state, gear_variable self, gear_variable element)

Inserts an element to the end of a list.

Parameters
state in The runtime context.
self in A variable whose value is a list.

If this is not a list, then the last error is set to GEAR_INVALID_VALUE.
element in A variable whose value will be inserted at the end of the list.

The list will be resized to accommodate the new element.

Since

0.1.0

void gear_list_set_element(gear_state* state, gear_variable self, int index, gear_variable input)

Stores a value into a list.

Parameters
state in The runtime context.
self in A variable whose value is a list.

If this is not a list, then the last error is set to GEAR_INVALID_VALUE.
index in A zero-based index into the list.
input in A variable whose value will be placed into the list at the given index.

The existing value at the given index will be replaced by the new value. If an out-of-bounds index is given, then the last error is set to GEAR_INVALID_OPERATION.

Since

0.1.0

void gear_list_get_element(gear_state* state, gear_variable self, int index, gear_variable output)

Retrieves the value of a list entry.

Parameters
state in The runtime context.

If this is not a list, then the last error is set to GEAR_INVALID_VALUE.
index in A zero-based index into the list.
output out A variable to assign the value of the list element at the given index.

If an out-of-bounds index is given, then the last error is set to GEAR_INVALID_OPERATION.

int gear_list_get_length(gear_state* state, gear_variable self)

Returns the number of elements in a list.

Parameters
state in The runtime context.
self in A variable whose value is a list.

If this is not a list, then the last error is set to GEAR_INVALID_VALUE.
Returns The number of elements in the list. The value will never be negative. 
// This example demonstrates retrieving the length of a list.

gear_state *state;  // Assume its initialized.
gear_variable list; // Assume it refers to a list.

int list_length = gear_list_get_length(state, list);

// Do something with the length of the list...

void gear_list_for_each(gear_state* state, gear_variable list, gear_for_each_element callback, void* user_data)

Iterates over each element of a list.

Parameters
state in The runtime context.
list in A variable whose value is a list.

If this is not a list, then the last error is set to GEAR_INVALID_VALUE.
callback in A function to invoke for each element of the list. If the list is empty, then this function will not be invoked.
user_data in A user pointer to give to the callback function; this can be NULL. 
// This example demonstrates iterating over a list.

gear_state *state;  // Assume its initialized.
gear_variable list; // Assume it refers to a list.

int iter_func(gear_state *state, void *user_data, gear_variable element, int index)
{
    printf("[%d] = %s\n", index, gear_to_string(element));
    return 0;
}

gear_list_for_each(state, list, iter_func, NULL);

Since

0.1.0
