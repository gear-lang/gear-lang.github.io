void gear_assign(gear_state* state, gear_variable to, gear_variable from)

Assigns the value of one native variable to another.

Parameters
state
to in The target of the assignment.
from in The source of the assignment.

Native variables should never be copied, inspected, or assigned directly. See the example below for how this function should be used.

// This example demonstrates assignment.
// Assume these variables are initialized.
gear_state *s;
gear_variable x, y;

x = y; // Incorrect

gear_assign(s, x, y); // Correct

gear_bool gear_equal(gear_state* state, gear_variable LHS, gear_variable RHS)

Performs an identity comparison between two variables.

Parameters
state in The runtime context.
LHS in The variable on the left-hand side of the comparison.
RHS in The variable on the right-hand side of the comparison.
Returns #gear_true if the variables refer to the same value.

Do not confuse this with an equality comparison. An identity comparison is only truthy when the variables reference the same underlying value.
