This page defines the complete compiler interface. Click on a symbol to visit its module and read its documentation.

The compiler interface includes everything for compiling individual Gear source files and modules, type checking, and parsing and scanning source code. If you wish to run code see gear_runtime.h.

The compiler symbols are defined in the libgear and libgearc. You must link with exactly one of them otherwise you will receive undefined or duplicate symbols.
