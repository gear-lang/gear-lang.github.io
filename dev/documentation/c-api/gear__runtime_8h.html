gear_runtime.h file

Complete interface definitions for the runtime.

Contents

  • Reference
    • Typedefs

This page defines the complete runtime interface. Click on a symbol to visit its module and read its documentation.

The runtime includes the virtual machine, garbage collector, and standard library; everything needed to execute Gear code. If you wish to compile code see gear_compiler.h.

The runtime symbols are defined in the libgear and libgearrt. You must link with exactly one of them otherwise you will receive undefined or duplicate symbols.
