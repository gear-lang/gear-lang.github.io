gear.h file

Imports both runtime and compiler interfaces.

Contents

  • Reference
    • Functions

This page defines interfaces that are only available when consuming both the runtime and compiler together. Click on a symbol to visit its module and read its documentation.

gear.h includes both gear_runtime.h and gear_compiler.h.

The runtime symbols are defined in libgear. Consumers of this header must link against libgear and not libgearrt or libgearc.
