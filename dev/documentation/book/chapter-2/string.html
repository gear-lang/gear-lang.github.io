3.3 String

Strings

Strings are immutable sequences of UTF-8 encoded unicode code points. They can be written with single or double quotes. There is no behavioral difference between them. Which one you use is personal preference.

"This is a double quoted string"
'This is a single quoted string'

Since everything is an object in Gear that means methods can be called on them directly. All of the objects provided by the standard library can be converted to strings using the toString method.

"Hello" concat(" world!")

Escaped Strings

Escapes sequences can be used to substitute a character with another character that may be more cumbersome to represent directly. Gear supports many of the same escape sequences available in C and Java.

In the follow example the character sequence \n will be substituted for the byte 0x0A.

"Escaped string \n split by a new line"

All of the escape sequences supported by Gear are shown in the following table.

Escape Sequence Character represented
\a Alert (beep, bell)
\b Backspace
\f Formfreed
\n Newline
\r Carriage return
\t Horizontal tab
\v Vertical tab
\ Backslash
\' Single quotation mark
\" Double quotation mark
\0cnnn The byte nnn interpreted as an octal
\0bn... The byte n... interpreted as a binary digit
\0xhh... The byte hh... interpreted as a hexadecimal number
\0Uhhhhhhhh Unicode code point where h is a hexadecimal digit
\0uhhhh Unicode code point below 10000 hexidecimal
\${expression} Disables string interpolation for expression

Unescaped Strings

Unescaped strings or raw strings are strings where the substitution of escape sequences has been disabled. They can be written by prefixing a string with an @.

The following example demontrates this feature for a double quoted string, but the behavior is the same for a single quoted string.

@"Unescaped string with a literal backslash 'n' at the end \n"

String Interpolation

String interpolation can be used to evaluate any arbitrary expression. String interpolation only works for escaped strings.

"The sum of 2 + 2 is ${2 + 2}"
