3. Builtin Types

Builtin Types

Gear includes a small set of types out of the box. This is because Gear was primarily designed as an embeddable programming language and it makes no assumptions about the needs of the application that hosts it. Embedders are expected to provide their own libraries for their needs.

This chapter explains Gear's builtin types at a high level. For an in-depth look at the entire standard library see the standard library documentation.

What follows is the complete list of Gear's builtin types:

  • Object
  • Number
  • Boolean
  • String
  • List
  • Map

Package Manager

If you need additional libraries, then use Gear's package manager gear up to acquire them. This is especially useful if Gear is used standalone (e.g. from a non-embedded context).
