3.3 List

Lists

Lists are ordered sequences of values (also known as "elements"). They can be written with bracket literals and can contain any mix types. The snippet below creates a mixed-type list of 3 elements.

[1, true, "Hello, World!"]

The empty list can be created with empty brackets [].

List Type

You can also create lists using the List type directly, but the convention is to use the literal syntax whenever possible.

let fruit = List new()
fruit push('apple')
fruit push('grape')
fruit push('watermelon')

Accessing Elements

List values can be randomly accessed using bracket notation. They are indexed starting from one where index 1 is the first element, index 2 is the 2nd element, and so on. Negative indices access elements from the end were index -1 accesses the last element, index -2 accesses the 2nd to last element, and so on.

let fruit = ['apple', 'grape', 'watermelon']
let apple = fruit[1]
let watermelon = fruit[-1]

Accessing index 0 will raise an "index out of bounds" exception.

In practice it is recommended to use the first and last messages to access the beginning and end of the list.

Iteration

You can iterate over a list using a for loop.

let fruit = ['apple', 'grape', 'watermelon']
for (element in fruit)
    print(element)
end

Alternatively, you can use messages like map and forEach to iterate.

['apple', 'grape', 'watermelon'].forEach(print)
