3.3 Dictionary

Dictionary

Dictionaries are unordered key/value pairs. Keys and values can be any type.

They can be written with curly bracket literals. The snippet below creates a dictionary using the literal syntax.

{'apple': 'red', true: 1, 42: 'answer to life'}

The empty dictionary can be created with empty curly brackets {}.

Dict Type

You can also create dictionaries using the Dict type directly, but the convention is to use the literal syntax whenever possible.

let age = Dict new()
age insert('Jill', 23)
age insert('Bob', 51)
age insert('Sue', 17)

Lookup

Dictionary values can be accessed using theirs via bracket notation. Accessing a key that does not exist when using bracket notation returns nil.

let fruit = ['apple', 'grape', 'watermelon']
let apple = fruit[1]
let watermelon = fruit[-1]

The bracket notation is identical to the lookup message. In practice it is recommended to use the bracket notation when looking up keys.

You can also use the lookup! message to return a value for a given key. The behavior of this message is the exact same as bracket notation, but if the key does not exist, then a "missing key" exception is raised.

To check if a key exists, use the hasKey message. It returns the boolean true if the key exists and false if it does not.

Iteration

You can iterate over a list using a for loop.

let fruit = ['apple', 'grape', 'watermelon']
for (key, value in fruit)
    print("key: ${key} value: ${value}")
end

Alternatively, you can use messages like map and forEach to iterate.

['apple', 'grape', 'watermelon'].forEach(
    fn(key, value)
        print("${key} = ${value}")
    end
)
