Examples at a Glance

Gear has numerous features, this page demonstrates some of them.

Checkout Gear's syntax reference for individual features.

Clean imperative syntax

Understandable at a glance.

fn factorial(n)
    if (n < 1)
        ret 1
    else
        ret n * factorial(n - 1)
    end
end
Message oriented

Create prototypical objects and send messages between them.

let rocket = Rocket new()
rocket.pilot = 'Neil Armstrong'
rocket blastOff()
Functional

Use closures and higher order functions.

let fruit = ['apple', 'watermelon', 'banana']
fruit sort(fn(x,y) x length() <=> y length())
Concurrent

Use coroutines and green threads. The example demonstrates how easy coroutines.

let constants = coroutine(fn()
    yield 3.14159
    yield 2.71828
end)
let pi = constants resume()
let e = constants resume()
Embeddable

Easily embed Gear inside your solution. The C snippet demonstrates just how easy it is.

#include <gear/gear.h>
int main(void)
{
    gear_state *s = gear_new();
    gear_eval(s, "print('Hello, World!'");
    gear_delete(s);
}
